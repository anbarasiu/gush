# Gush - Writing Inspiration Prompts

**Live Site:** [gush.anbuu.in](https://gush.anbuu.in)

Gush is a writing inspiration prompts app by Anbarasi U (Anbu) to help writers overcome writer's block with creative prompts and ideas.

## About

Gush provides creative writing prompts to inspire writers, poets, and storytellers. Whether you're facing writer's block or just looking for fresh ideas, Gush offers diverse prompts to get your creative juices flowing.

### Features

- **Random Writing Prompts**: Generate fresh inspiration instantly
- **Creative Categories**: Diverse prompts for different writing styles
- **PWA Support**: Install as a Progressive Web App for offline access
- **Clean Interface**: Distraction-free experience focused on creativity
- **Share Prompts**: Easy sharing of prompts across social media

## SEO Enhancements

This app includes comprehensive SEO optimization:

- **Meta Tags**: Enhanced title, description, and keywords
- **Open Graph**: Rich previews on social media
- **Twitter Cards**: Beautiful link previews
- **JSON-LD Structured Data**: WebApplication and Person schemas
- **AI Crawler Permissions**: GPTBot, Claude-Web, PerplexityBot, etc.
- **Sitemap**: Auto-generated by Gatsby
- **PWA Manifest**: App installation support

## Technology Stack

- **Gatsby** - React-based static site generator
- **React** - UI framework
- **React Helmet** - SEO and meta tag management
- **Sass** - CSS preprocessing
- **PWA** - Progressive Web App capabilities
- **Segment** - Analytics integration

## Local Development

```bash
# Clone the repository
git clone https://github.com/anbarasiu/gush.git
cd gush

# Install dependencies
npm install

# Run development server
npm start

# Build for production
npm run build
```

The dev site will be available at `http://localhost:8000`

## Deployment

```bash
npm run build
```

The production build will be in the `public/` directory, ready to deploy to GitHub Pages or any static hosting service.
